주소록 관리
 
그룹 관리
  '그룹 추가' 버튼을 눌러 그룹을 추가할 수 있습니다.
  그룹을 편집또는 삭제하려면 해당 그룹을 선택 후 오른쪽 버튼을 눌러 '편집' 또는 '삭제'를 클릭하세요.
  선택한 그룹의 명단 모두를 수신자 목록에 추가하려면 해당 그룹을 선택 후 오른쪽 버튼을 눌러 '이 그룹의 명단을 수신자 리스트에 추가'를 클릭하세요.

개인 관리
  주소를 추가하려면 '개인 추가' 버튼을 눌러 개인 주소를 추가할 수 있습니다.
  내용을 편집하려면 항목을 선택한 후 글자를 치거나, F2를 눌러 편집할 수 있으며, 오른쪽 버튼을 눌러
    '편집'을 클릭해도 됩니다.
  내용을 삭제하려면 지울 행들을 선택한 후 DEL 키를 누르거나 오른쪽 버튼을 눌러 '삭제'를 클릭해도 됩니다.
  하나 또는 다중 선택한 항목을 수신자 목록에 추가하려면 오른쪽 버튼을 눌러 '수신자 리스트에 추가'를 클릭하면 됩니다.
  선택한 주소를 다른 그룹으로 이동하려면 오른쪽 버튼을 눌러 '이동'을 펼친 후 이동될 그룹을 선택하십시오.

수정사항 적용하기
  주소록에서 편집한 작업은 바로 적용되지 않고 사용자가 편집한 내용을 기억하고 있다가 '저장'버튼을 누르면 DACOM SMS메신저 서버로 편집 사항을 알립니다.
    따라서 편집한 것을 서버에 저장하려면 반드시 이 버튼을 눌러야 합니다.

 
수정사항 취소하기
  편집한 내용을 모두 무시하도록 할 수 있습니다. '편집 취소' 버튼을 누르면 편집을 모두 취소하며 서버에서 기존의 주소록을 다시 불러옵니다.
