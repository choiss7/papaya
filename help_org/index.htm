프로그램 특징
 
설치 사양
  DACOM SMS메신저 제품군은 Microsoft Windows 98 이상의 운영체제와 Microsoft Internet Explorer 5.0이상이 설치되어 있는 컴퓨터에서 사용 가능합니다.
  프로그램을 실행하기 위해서는 반드시 인터넷 통신이 가능한 상태여야 합니다.
  엑셀프로그램이 반듯이 설치되어 있어야 하며 설치되어 있지 않을 경우에는 엑셀 관련 기능을 사용할 수 없게 됩니다.(Excel 2000 이상 설치 필수)
엑셀의 고유기능 사용
  필터링, 행 삽입, 행 추가, 선택된 모든 행 삭제, 모든 행 삭제, 편집, 복사, 잘라내기, 붙여넣기, 지우기 등 ( 단, 숨기기 기능 제외 )
다량 동시 전송
  DACOM SMS메신저 제품군은 한번에 여러사람에게 메시지를 보낼 수 있도록 제작되었으며
최대 65,000건 이상의 동시 전송 및 예약 전송이 가능합니다.
엑셀 파일 불러오기
  DACOM SMS메신저 Pro에서는 엑셀 파일을 읽어 각 시트별로 내용을 추출하여 수신자 목록에 나열 또는 주소록 목록에 추가할 수 있습니다.
    자세한 내용은 [엑셀 활용]을 참고하세요.
메일 머지
  메일머지 기능을 이용하여 보다 다양하고 선택적인 메시지를 여러사람에게 보낼 수 있습니다.
    자세한 내용은 [메일머지]를 참고하세요.
클립보드 복사
  DACOM SMS메신저 제품군은 엑셀의 클립보드 형식을 읽어들일 수 있으며, 역으로 DACOM SMS메신저 제품군에서 작성된 데이터를 엑셀로 복사할 수도 있습니다.
  또한 탭과 엔터로 구분된 텍스트도 사용할 수 있습니다.
  현재 웹페이지의 테이블도 복사할 수 있게끔 개발중에 있으며 보다 자세한 내용은 [클립 보드]를 참고하세요.
