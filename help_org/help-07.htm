엑셀 파일 활용
 
시트 선택
  불러온 엑셀 파일 시트중에 하나를 클릭하면 오른쪽 리스트에 해당 시트의 데이터가 채워집니다.
데이터 편집
  '엑셀 파일에서 가져오기' 다이얼로그에서는 각 항목의 내용을 편집할 수는 없으며 행 삭제와 열 삭제만 가능합니다. 엑셀 파일은 읽기 전용으로 열기 때문에 여기서 삭제된 행과 열은 엑셀 원본 파일에 지장을 주지 않습니다.
  행을 삭제하기 위해서는 행번호를 선택(행번호 위에서 마우스 클릭, 드래그, Ctrl, Shift 조합 가능)한 후 오른쪽 버튼을 누르면 해당 기능이 나타납니다.
  열을 삭제하기 위해서는 열을 선택(A1, A2 등의 열에서 마우스 클릭, Ctrl, Shift 조합 가능)한 후 오른쪽
    버튼을 누르면 해당 기능이 나타납니다.
