기본 사용 방법
 
로그인
  DACOM SMS메신저에서 SMS 문자를 전송하려면 로그인을 해야 합니다.
  툴바에서 로그인 버튼을 클릭한 후   http://sms.dacom.net 에서 로그인할 때 사용하는 아이디와 패스워드를 넣고 확인을 누릅니다.
SMS 전송
  수신자 목록에 문자를 받을 명단을 작성하고, 메시지창에 보낼 문자를 작성한 후 '전송' 버튼을 누르면 메시지가 전송됩니다.

적용가능한 핸드폰번호 형식
  일반적으로 '이동통신사 번호+국번+번호" 의 형식으로 입력합니다.
예) 011-123-4567, 017 123 4567, 016-1234567, 0181234567, 010 1234567,  191234567
잘못된 예) 019)123-4567
수신목록 생성 방법
  수신목록에서 문자를 받을 사람들의 목록을 직접생성합니다.
  열기를 누르거나 주소록 관리에 등록되어 있는 그룹을 선택하여
수신목록에 옮겨와 목록을 생성합니다.
전송방법
  전송하기를 눌러서 전송창을 띄운 다음, 메시지창에 보낼 문자를 작성한 후 수신핸드폰번호의 필드를 선택하고 회신번호를 지정한 다음 '전송' 버튼을 누르면 메시지가 전송됩니다.
특수문자

이모티콘
  카테고리 별로 이모티콘을 선택하여 문자를 전송할 수 있습니다. 
 
  이모티콘의 등록과 수정이 용이 합니다 (내 이모티콘)

메일머지
  여러 사람에게 다른 내용의 메시지를 보낼 때,
특정 부분만 그 사람의 데이터에 맞는 문자를 만들어야 할 경우에 사용합니다
