SMS 전송
 
주소록에서 수신 목록 가져오기
  '주소록에서 가져오기' 버튼을 누르면 팝업 메뉴에 자신의 주소록에 있는 그룹 명단이 나타납니다.
  항목을 클릭하면 기존의 수신자 목록의 데이터는 모두 지워지고 메뉴에서 선택한 그룹의 명단이 리스트에 나타납니다.

 
엑셀 파일 불러오기
  DACOM SMS메신저에서는 엑셀 파일을 읽어 각 시트별로 내용을 추출하여 수신자 목록에 나열할 수 있습니다.
  '엑셀 파일 읽기'() 버튼을 누르면 파일을 고르는 대화상자가 나오며 여기서 엑셀 파일을 고르고 '열기' 버튼을 누릅니다. 왼쪽에서 원하는 시트를 선택하면 오른쪽 리스트에 시트 내용이 채워집니다.
    이 때, '확인'버튼을 누르면 대화상자가 닫히면서 수신자 목록의 모든 데이터를 삭제 후 엑셀 데이터가 채워집니다.

수신자 목록 관리
  수신자 목록에서 원하는 항목에 커서를 두고 글자를 치면 바로 내용을 편집할 수 있습니다.
    (행 삽입, 행 추가, 선택된 모든 행 삭제, 모든 행 삭제, 편집, 복사, 잘라내기, 붙여넣기, 지우기 등
    아래 이미지 참조)

보내기
  수신자 목록의 핸드폰에 메시지를 동시 전송합니다.
예약 설정
  '예약 설정' 버튼을 누르면 메시지를 지정한 날짜/시간에 예약 전송하도록 설정할 수 있습니다.

간격 설정
  '간격 설정' 버튼을 누르면 지정한 간격으로 메시지를 보낼 수 있도록 설정할 수 있습니다.

메일머지 조합
  '메일머지 조합' 버튼을 누르면 메일머지 메시지를 쉽게 조합 할 수 있습니다.
